### 11. Corrigir `create_chart()` parsing de ranges
- **Arquivo:** `scripts/office_mcp.py` (função `create_chart`, ~linhas 1254-1255)
- **Problema:** `str.replace()` destrói números de linha em colunas com 2+ letras (ex: "AB1:AB10" → "B10" em vez de "10"); `min_col` hardcoded em 1 e 2
- **Ação:** Usar regex para extrair partes corretamente:
  ```python
  import re
  def parse_range(ref):
      m = re.match(r'^([A-Z]+)(\d+):([A-Z]+)(\d+)$', ref)
      if not m: raise ValueError(f"Range inválido: {ref}")
      return m.group(1), int(m.group(2)), m.group(3), int(m.group(4))
  ```
  Converter letras de coluna para número com `openpyxl.utils.column_index_from_string()`
- **Verificação:** Testar com ranges: `B1:B10`, `AA1:AA10`, `AB1:AB10`, `ZA1:ZA10`
