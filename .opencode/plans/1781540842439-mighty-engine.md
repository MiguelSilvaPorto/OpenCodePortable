# Plano: Setup Automático no opencode.bat

## Objetivo

Modificar o `opencode.bat` para que **sempre** verifique e instale automaticamente as dependências necessárias a cada execução, não apenas na primeira vez.

## Problema Atual

```
Linha 141: if exist "%FIRST_RUN%" goto :start
```

Se o marker `.voice-setup-done` existir, TODO o setup é pulado. Em outro PC:
- Marker existe (copiado)
- Dependências NÃO existem (não copiadas)
- Resultado: MCP servers não funcionam, microfone some

## Solução

Criar uma rotina de **health check rápido** que roda a cada execução:

```
┌─────────────────────────────────────────────────┐
│  opencode.bat inicia                            │
├─────────────────────────────────────────────────┤
│  1. Verificar executável (já existe)            │
│  2. Verificar atualizações (já existe)          │
│  3. NOVO: Health check de dependências          │
│     ├── Scoop instalado?                        │
│     ├── whisper-cli no PATH?                    │
│     ├── sox no PATH?                            │
│     ├── Modelo Whisper existe?                  │
│     ├── Python no PATH?                         │
│     ├── Bibliotecas Python instaladas?          │
│     └── Se faltar ALGO → instalar automaticamente│
│  4. Corrigir caminhos MCP (já existe)           │
│  5. Iniciar OpenCode                            │
└─────────────────────────────────────────────────┘
```

## Mudanças no opencode.bat

### 1. Adicionar função `:health_check` (após linha 436)

```batch
:: ============================================
:: Sub-rotina: Health Check de Dependências
:: Verifica e instala automaticamente o que faltar
:: ============================================
:health_check
set "HEALTH_OK=1"

echo [HEALTH] Verificando dependencias...

:: 1. Scoop
where scoop >nul 2>&1
if %errorlevel% neq 0 (
    echo [HEALTH] Scoop nao encontrado. Instalando...
    powershell -Command "Set-ExecutionPolicy RemoteSigned -Scope CurrentUser -Force; [Net.ServicePointManager]::SecurityProtocol = [Net.SecurityProtocolType]::Tls12; Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression"
    if !errorlevel! neq 0 (
        echo [HEALTH] ERRO: Falha ao instalar Scoop
        set "HEALTH_OK=0"
    ) else (
        set "PATH=%USERPROFILE%\scoop\shims;%PATH%"
        echo [HEALTH] OK: Scoop instalado
    )
) else (
    echo [HEALTH] OK: Scoop
)

:: 2. whisper-cpp
where whisper-cli >nul 2>&1
if %errorlevel% neq 0 (
    echo [HEALTH] whisper-cli nao encontrado. Instalando...
    call scoop install whisper-cpp
    if !errorlevel! neq 0 (
        echo [HEALTH] ERRO: Falha ao instalar whisper-cpp
        set "HEALTH_OK=0"
    ) else (
        echo [HEALTH] OK: whisper-cpp instalado
    )
) else (
    echo [HEALTH] OK: whisper-cpp
)

:: 3. sox
where sox >nul 2>&1
if %errorlevel% neq 0 (
    echo [HEALTH] sox nao encontrado. Instalando...
    call scoop install sox
    if !errorlevel! neq 0 (
        echo [HEALTH] ERRO: Falha ao instalar sox
        set "HEALTH_OK=0"
    ) else (
        echo [HEALTH] OK: sox instalado
    )
) else (
    echo [HEALTH] OK: sox
)

:: 4. Modelo Whisper
if not exist "%MODELS_DIR%\ggml-base.bin" (
    echo [HEALTH] Modelo Whisper nao encontrado. Baixando...
    if not exist "%MODELS_DIR%" mkdir "%MODELS_DIR%"
    curl -L -o "%MODELS_DIR%\ggml-base.bin" "https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-base.bin"
    if !errorlevel! neq 0 (
        echo [HEALTH] ERRO: Falha ao baixar modelo
        set "HEALTH_OK=0"
    ) else (
        echo [HEALTH] OK: Modelo Whisper baixado
    )
) else (
    echo [HEALTH] OK: Modelo Whisper
)

:: 5. Python
where python >nul 2>&1
if %errorlevel% neq 0 (
    echo [HEALTH] Python nao encontrado. Instalando...
    call scoop install python
    if !errorlevel! neq 0 (
        echo [HEALTH] ERRO: Falha ao instalar Python
        set "HEALTH_OK=0"
    ) else (
        echo [HEALTH] OK: Python instalado
    )
) else (
    echo [HEALTH] OK: Python
)

:: 6. Bibliotecas Python
python -c "import openpyxl, docx, pptx, mcp, win32com.client, psutil, formulas, msal, pdf2image, lxml" >nul 2>&1
if %errorlevel% neq 0 (
    echo [HEALTH] Bibliotecas Python incompletas. Instalando...
    python -m pip install --upgrade pip >nul 2>&1
    python -m pip install openpyxl python-docx python-pptx pywin32 mcp psutil formulas msal pdf2image lxml >nul 2>&1
    python -c "import openpyxl, docx, pptx, mcp, win32com.client, psutil, formulas, msal, pdf2image, lxml" >nul 2>&1
    if !errorlevel! neq 0 (
        echo [HEALTH] ERRO: Falha ao instalar bibliotecas
        set "HEALTH_OK=0"
    ) else (
        echo [HEALTH] OK: Bibliotecas Python instaladas
    )
) else (
    echo [HEALTH] OK: Bibliotecas Python
)

:: 7. Azure CLI (opcional)
where az >nul 2>&1
if %errorlevel% neq 0 (
    echo [HEALTH] Azure CLI nao encontrado (opcional). Instalando...
    call scoop install azure-cli
    if !errorlevel! neq 0 (
        echo [HEALTH] AVISO: Falha ao instalar Azure CLI (opcional)
    ) else (
        echo [HEALTH] OK: Azure CLI instalado
    )
) else (
    echo [HEALTH] OK: Azure CLI
)

echo [HEALTH] Verificacao concluida.
goto :eof
```

### 2. Modificar fluxo principal (linha 141)

**ANTES:**
```batch
if exist "%FIRST_RUN%" goto :start
```

**DEPOIS:**
```batch
:: SEMPRE executar health check (removido bypass por marker)
call :health_check

:: Se.marker não existe, mostrar setup inicial completo
if not exist "%FIRST_RUN%" (
    echo.
    echo ============================================
    echo   Configuracao Inicial
    echo ============================================
    echo.
    REM ... (código existente do setup inicial)
)
```

### 3. Manter compatibilidade

O marker `.voice-setup-done` continua existindo para:
- Saber se o setup inicial completo já foi executado
- Evitar mostrar a mensagem "Configuração Inicial" toda vez

Mas o health check roda **sempre**, independente do marker.

## Vantagens

1. **Portabilidade total** - Funciona em qualquer PC Windows
2. **Zero config manual** - Usuário só roda `opencode.bat`
3. **Idempotente** - Pode rodar várias vezes sem quebrar
4. **Rápido quando OK** - Health check leva ~0.5s
5. **Automático quando falta** - Instala tudo sozinho
